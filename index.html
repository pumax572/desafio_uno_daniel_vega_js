Bucles con JavaScript

Daniel Maximiliano Vega

Práctica con bucles: A veces es necesario que ciertas partes de un código se ejecuten varias veces.
Hay tres elementos que controlan el flujo de ejecución de un bucle.

  1. Inicialización: Fija los valores con los que ejecutamos el bucle.
  2. Condición de Permanencia: Cuanta las veces que queremos ejecutar el bucle.
  3. Actualización de la variable que controla la condición.
